# Web scraping et collecte de données

## Objectif
Collecter automatiquement des données textuelles à partir de pages web
en utilisant des techniques de web scraping.

## Méthodologie
- Envoi de requêtes HTTP avec la bibliothèque `requests`
- Analyse du code HTML avec BeautifulSoup
- Parcours de plusieurs pages
- Extraction et structuration des données collectées

## Résultats
Le projet montre comment construire un pipeline simple de collecte de données
web pour une utilisation ultérieure en analyse ou en machine learning.

## Compétences mobilisées
- Web scraping
- Collecte et préparation de données
- Manipulation de données non structurées
