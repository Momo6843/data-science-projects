# Prédiction du churn client

## Objectif
Prédire la probabilité qu’un client d’un opérateur télécom résilie son abonnement
à partir de ses caractéristiques et de son historique de services.

## Données
Jeu de données "Telco Customer Churn" contenant des informations
démographiques, contractuelles et comportementales des clients.

## Méthodologie
- Nettoyage et préparation des données
- Gestion du déséquilibre des classes
- Analyse exploratoire des données (EDA)
- Ingénierie des variables
- Entraînement de modèles de classification

## Résultats
Les modèles obtenus permettent d’identifier les principaux facteurs
influençant le churn client et d’obtenir des performances prédictives satisfaisantes.

## Compétences mobilisées
- Analyse de données réelles
- Classification sur données déséquilibrées
- Interprétation métier des résultats
